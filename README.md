## ✨ Features

*   **Premium Glassmorphic UI**: Vibrant, responsive dark theme styled with custom HSL variables.
*   **Viewport Centering Navigation**: 3-column CSS Grid navbar that perfectly centers menu items horizontally.
*   **Framer Motion Animations**: Micro-interactions, spring animations, and smooth transitions on hover/scroll.
*   **Dynamic Typewriter**: Typewriter effect displaying key title roles in the Hero section.
*   **Contact Form with Resend**: Lightweight backend email integration fallback to native mail client links.

## ⚙️ Local Development

1.  **Clone & Navigate**:
    ```bash
    git clone https://github.com/om0710/my-portfolio.git
    cd my-portfolio
    ```
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Run Development Server**:
    ```bash
    npm run dev
    ```
    Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
4.  **Production Build**:
    ```bash
    npm run build
    ```

---

## 🚀 Deploy on Vercel

This repository is optimized for Vercel out of the box:
1. Log in to [Vercel](https://vercel.com/) and click **Add New > Project**.
2. Import this repository: `my-portfolio`.
3. Keep default settings and click **Deploy**. Vercel will automatically build the Next.js application!
